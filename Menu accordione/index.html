Торт для торжественной даты?

Можно выбрать торт с индивидуальным декором, ручной росписью, в несколько ярусов в зависимости от желаемых размеров.

Торт для небольшого домашнего праздника?

Можно выбрать торт с облегченным декором, весом от 2кг в один ярус.

Десерт для ребенка?

Можно выбрать кейк-попсы для кенди-бара, капкейки либо торт с детским тематическим декором.

Без повода?

Тогда любой десерт, на котором остановится ваш взгляд. Фантазию и желания не ограничиваем ;) !
